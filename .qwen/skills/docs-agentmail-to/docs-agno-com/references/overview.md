# Overview

> Process and generate images, audio, video, and files with agents and teams.

Agno enables agents and teams to process and generate text, images, audio, video, and files.
Build sophisticated AI applications with Agno to address a wide range of multimodal AI use cases including these:

* Image generation and analysis with contextual responses
* Audio transcription and generation
* Video processing and generation
* Understanding documents
* Multi-tool and multi-turn interactions

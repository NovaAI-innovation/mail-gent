# Delete Component

> Delete a component and all its config versions.

## Path Parameters

| Parameter      | Type     | Description      |
| -------------- | -------- | ---------------- |
| `component_id` | `string` | The component ID |

## Response `204`

No content. The component and all its config versions are deleted.

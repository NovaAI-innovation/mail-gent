# Delete Config Version

> Delete a specific config version.

## Path Parameters

| Parameter      | Type     | Description        |
| -------------- | -------- | ------------------ |
| `component_id` | `string` | The component ID   |
| `version`      | `int`    | The version number |

## Response `204`

No content. The config version is deleted.
